Privacy Policy

Last Updated: Today

Escape Button is designed to provide a quick shortcut to close all tabs, clear browsing history (except for whitelisted sites), and open a Safe Site. This extension focuses only on this single purpose and does not track, store, upload, or share any user data.

Information We Access

Escape Button only accesses one type of browser data:

  • Web History – accessed only to delete non-whitelisted history entries when the user activates the shortcut.

No other information is accessed, collected, or processed.

Information We Do NOT Collect

Escape Button does NOT collect, store, or transmit any of the following:

  • Personal information (name, email, age, address…)
  • Passwords, authentication info, or credentials
  • Financial or payment information
  • Location or IP data
  • Health data
  • Messages, communications, or website content
  • User activity (mouse, keystrokes, clicks, etc.)

Data Storage

The extension only stores your Safe Site and Whitelist settings locally using Chrome’s storage API. This data never leaves your device.

No Data Sharing

Escape Button does not send any data to developers, servers, analytics tools, advertisers, or third parties. Nothing is uploaded or transferred.

Remote Code

Escape Button does not use remote code, external scripts, CDNs, or eval(). All code is packaged inside the extension.

Changes to This Policy

Any updates to this Privacy Policy will be posted on this page.
